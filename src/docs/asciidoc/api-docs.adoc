```
토큰에 수정이 있을 경우/ 토큰 조작, 만료, 서명실패, 디코딩 실패시 같은 status이고, message만 다름
{
    "status": 403,
    "message": "Malformed Token",
    "data": false
}
```

```
토큰이 없거나 너무 짧은 경우
{
    "status": 401,
    "message": "Token is null or too short",
    "data": false
}

```
